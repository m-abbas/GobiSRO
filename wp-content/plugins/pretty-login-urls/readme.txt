Use pretty urls like /login instad of wp-login.php .With this plugin you can change the login or register url to whatever you want

== Description ==

This simple plugin use wordpress rewrite api to change login and register url to whatever you want. On activation by default will change wp-login.php to "/login" and wp-register.php to "/register". You can change these on the settings page.

== Installation ==

1. Unzip and Upload the directory 'pretty-urls' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to Settings; You'll see a new menu link called 'Pretty Login Url'.
